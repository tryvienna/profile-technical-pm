# 🎯 Technical Product Manager

> What should we build next? Let the data decide.

A Vienna content profile for technical product managers. Includes quick actions for feature prioritization, PRD writing, RICE scoring, sprint summaries, release notes, and funnel analysis.

## What's included

- **Quick Actions** — Prioritize features with data, write PRDs, generate RICE scores, summarize sprints, draft release notes, analyze adoption funnels
- **Skills** — Getting started guide for Vienna profiles
- **Plugins** — Linear for roadmap and sprint management

## Install

Fork this profile in Vienna:

```
vienna://profile/github.com/tryvienna/profile-technical-pm
```

Or go to **Settings → Profiles → Fork** and paste:
```
https://github.com/tryvienna/profile-technical-pm.git
```

## Customize

After forking, this profile lives at `~/.vienna/profiles/technical-pm/`. Edit any file to make it yours — add quick actions, swap plugins, tweak skills.

## Share

Push your customized profile to your own GitHub repo and share the `vienna://profile/...` link. Others can fork your version in one click.
